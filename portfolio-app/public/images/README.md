# Images Folder

Place your images in the following folders:

## Folder Structure

```
public/images/
├── logo/
│   └── logo.png (or .svg, .jpg - Your logo image for navbar and footer)
├── hero/
│   └── hero-bg.jpg (or .png - Hero section background image)
└── profile/
    └── profile.jpg (or .png - Your profile photo for the About section)
```

## Image Specifications

### Logo (`logo/logo.png`)
- Recommended size: 200x50px or similar aspect ratio
- Formats: PNG (with transparency), SVG, or JPG
- Used in: Navbar and Footer

### Hero Background (`hero/hero-bg.jpg`)
- Recommended size: 1920x1080px or higher
- Format: JPG or PNG
- Used as: Full-screen background in the hero section

### Profile Photo (`profile/profile.jpg`)
- Recommended size: 400x400px or square aspect ratio
- Format: JPG or PNG
- Used in: About section

## How to Change Images

Simply replace the image files in their respective folders with the same filename. The changes will appear automatically when you refresh your browser.

## Supported Formats

- PNG (recommended for logos with transparency)
- JPG/JPEG (recommended for photos)
- SVG (recommended for logos)
- WebP (modern, efficient format)

## Note

Make sure to keep the filenames the same, or update the file paths in `src/App.tsx` if you use different names.
