# Resume Folder

Place your resume PDF file in this folder.

## Instructions

1. **File Name:** Name your resume file as `resume.pdf`
2. **Location:** Place it in `public/resume/resume.pdf`
3. **Format:** PDF format is recommended for best compatibility

## Supported Formats

- PDF (recommended)
- DOCX (Word document)
- TXT (Plain text)

## How to Update

Simply replace the `resume.pdf` file in this folder with your updated resume. The download link will automatically use the new file.

## Note

If you use a different filename or format, make sure to update the file path in `src/App.tsx` at the "Download Resume" button.
